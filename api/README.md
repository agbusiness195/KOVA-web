# AI Chat API

The help widget chat runs on this serverless function. To enable it:

1. **Deploy to Vercel** — Connect your repo and deploy.
2. **Add `GROQ_API_KEY`** (free) — Sign up at [console.groq.com](https://console.groq.com), create an API key, add it to Vercel: Project → Settings → Environment Variables.
3. **Redeploy** — The chat will work on your live site.

**Free:** Groq has a free tier, no credit card required. Use `GROQ_API_KEY` in Vercel env.

**Optional:** `OPENAI_API_KEY` also works if you prefer OpenAI.
